# Options Risk Education

This project may reuse selected concepts from `himself65/trade-skills` only as
research and education material.

Allowed uses:

- payoff/risk concept explanations
- volatility-regime awareness as educational context
- common options pitfalls and case-study warnings
- payoff charts, max loss, max profit, and breakeven calculations only when the
  user supplies the exact option legs

Not allowed:

- suggested option trades
- recommended strikes
- recommended expirations
- entry or exit instructions
- position sizing advice
- executable order payloads
- broker integration or order routing

If an options-payoff feature is added later, it must:

- be clearly labeled educational only
- require user-supplied legs or clearly labeled example legs
- explain assumptions and limitations
- avoid any buy/sell/hold recommendation language
- never place or prepare orders
